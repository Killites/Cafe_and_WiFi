# ☕ Cafe & WiFi Finder 💻

A simple and aesthetic web application to find cafes with WiFi and power for remote working. Built using Flask and SQLite, this project allows users to view, add, and delete cafes with ease.

---

## 🚀 Features

- 📍 View cafes with location links (Google Maps)
- ➕ Add new cafes
- ❌ Delete existing cafes
- ☕ Clean coffee-themed UI
- 🌿 Minimal aesthetic design

---

## 🛠️ Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
